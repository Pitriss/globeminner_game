Globeminner game
================

Game based on old minetest_game with added mods listed on globeminner server.
This game is *NOT* exact game used on globeminner servers. It is based on mod listing from these servers,
assembled mod by mod. It also contains few more mods, tweaks to provide more smooth gameplay.
This game is for minetest 0.4.6 and may contain obsolete and buggy mods.
